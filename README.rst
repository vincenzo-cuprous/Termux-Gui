Termux Gui
==========

- It will allows you to change your termux into a linux with full gui.

Install Command
===============

.. code-block:: bash

   curl https://raw.githubusercontent.com/vincenzo-cuprous/Termux-Gui/main/freebsd | sh

Usage
=====

- On Bsd

.. code-block:: bash

   vnc-on
   vnc-off
